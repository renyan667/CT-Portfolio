My Portfolio Website Project

Reflection

This is my first computing project ever! In this project, I first learned about the basics of coding like what HTML, CSS and JS stands for. Then, I got started coding in HTML and CSS with the help of websites like W3schools and freecodecamp. My class also did a few lessons where we followed along the teacher as we started websites in VS code. Then, I downloaded and made my first accounts on Github and learned all about repositories, pull requests and commits. Because of this, I learnt to keep a record and save all my commits, showing my website building process. But before I got to coding the website, first I sketched some ideas and took inspiration from other portfolio websites on the Internet. After this point, I got to actually coding my website. First I learnt about flexbox, how it works and how to use it to make a website that would be accessible through multiple screen dimensions. Then, I chose the typography, background colours and main theme for my website. In this process, I learnt to chose my own style of websites and to transfer my ideas into a design. For my website, I chose a simple greyscale style be more legible but I still used a large eye-catching title. I used buttons to lead the user to the projects and I made them get bigger on hover. In the middle of the pages, I have a cinematic photo of some mountains that move as you scroll. In this project, I faced many problems like typos resulting to broken code, code just not working and I was able to adapt and overcome to these problems. My goal for the future is to be able to be able to code more visually appealling and interactive websites and to resolve problems faster. I would also want to code a hamburger menu for users on a phone in the future
