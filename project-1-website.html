Reflection

This is my first computing project ever! In this project, I first learned about the basics of coding like what HTML, CSS and JS stands for. Then, I got started coding in HTML and CSS with the help of websites like W3schools and freecodecamp. My class also did a few lessons where we followed along Mr Lo Russo as we started websites in VS code. Then, I downloaded and made my first accounts on Github and learned all about repositories, pull requests and commits. But before I got to coding the website, first I sketched some ideas and took inspiration from other portfolio websites on the Internet. After this point, I got to actually coding my website. First I learnt about flexbox, how it works and how to use it to make a website that would be accessible through multiple screen dimensions. Then, I chose the typography, background colours and main theme for my website.
